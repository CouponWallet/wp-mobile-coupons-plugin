== Description ==

Add this mobile coupons plugin to add coupon management to Wordpress. Coupons may be printed or saved onto the customer's mobile phone for use in-store at the point-of-sale.

Use coupons for:
*    marketing
*    sharing on social media
*    lead capture
*    list building
*    driving in new customers

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/wp-mobile-coupons-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Mobile Coupons screen to configure the plugin
4. Visit the Mobile Coupons section to view, edit, and add Mobile Coupons to your website 
5. Add your Coupon Wallet API key in the settings for additional functionality

== Frequently Asked Questions ==

= Do I require a Coupon Wallet API key? =

No. You may use this plugin to manage coupons on your website without a Coupon Wallet account, however, with a key you will receive **additional features** such as tracking, list building and integration with point-of-sale.
